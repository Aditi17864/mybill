# **App Name**: RetailFlow

## Core Features:

- Secure Authentication: Enable login via mobile OTP verification and Google Sign-In for admin access.
- Shop Selection: Allow the user to select either 'Kapish Photo Frame' or 'Sunny Watch Center' for the current billing session and persist the shop selection for billing.
- Invoice Generation: Dynamically generate invoice in real-time, including customer details, items purchased, prices, and auto-calculated total.
- Payment Options: Enable payment via Cash or UPI, generate a dynamic QR code for UPI payments, and update payment status.
- Invoice Confirmation: Generate a shareable digital invoice including shop details, customer details, itemized purchases, and payment details.
- Real-time Dashboard: Display key real-time metrics such as total sales today, cash vs. UPI totals, the number of bills generated, and shop-wise sales breakdowns.
- Monthly Summaries: Provide total monthly revenue and daily sales graphs for comprehensive business insights.

## Style Guidelines:

- Primary color: Soft blue (#90AFC5) to provide a calm, professional feel.
- Background color: Light gray (#F0F4F8) to maintain a clean, light theme.
- Accent color: Deep orange (#D46A4E) to highlight key actions and CTAs.
- Body and headline font: 'PT Sans' sans-serif for a modern yet accessible feel. 
- Use simple, professional icons for navigation and shop-specific branding.
- Employ a card-based layout to clearly separate the sections and data.
- Apply smooth transitions on page navigation and data updates.
